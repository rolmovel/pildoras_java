# Ejercicio: Sistema de e-commerce

## Requisitos

 
- Clases: `Product`, `ShoppingCart`, `ECommerceSystem`
- `Product`: SKU, nombre, precio
- `ShoppingCart`:
 
  - Debe gestionar múltiples productos
  - Calcular total
  - Aplicar descuentos
- `ECommerceSystem`:
 
  - Gestión de inventario
  - Procesar órdenes
