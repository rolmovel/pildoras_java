# Ejercicio: Modelado de vehículo

## Requisitos

- Crear una clase `Car` que represente un vehículo

- Atributos:
  - `marca` (string)
  - `velocidad` (entero)

- Métodos:
  - `acelerar(int incremento)`: aumenta la velocidad
  - `frenar(int decremento)`: reduce la velocidad

- Restricciones:
  - La velocidad nunca puede ser negativa
  - Los métodos solo deben actuar con valores positivos
