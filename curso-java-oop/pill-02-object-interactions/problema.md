# Ejercicio: Sistema de biblioteca

## Requisitos

- Crear clases: `Library`, `Book`, `Author`
- `Book` debe contener:
  - Título
  - Referencia a objeto `Author`
  - Año de publicación
- `Library` debe:
  - Gestionar colección de libros
  - Permitir búsquedas por autor/título
  - Prestar/devolver libros
